# Vagrant LAMP stack

This provides a Hetzner-ish shared hosting vagrant box for
improved development-production parity.

### Whats on this box?

- Debian 7.8 (wheezy)
- PHP 5.4
- Apache 2.2
- Mysql 5.5

### Setup

1. Create a folder for your sites (e.g. ~/Sites)
2. Checkout this repo
3. run `SITES_DIR=~/Sites sudo make install`
